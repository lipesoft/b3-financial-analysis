# B3 Financial Analysis

Dashboard interativo para análise de dados financeiros de empresas listadas na B3.

## Sobre o projeto

Este projeto utiliza Python para analisar dados financeiros de empresas da bolsa brasileira.
O dashboard permite visualizar receita, lucro, crescimento financeiro e comparar empresas através de gráficos e indicadores.
A aplicação foi construída utilizando **Streamlit**, permitindo interação direta com os dados.

## Funcionalidades

* Visualização de dados financeiros das empresas
* Gráficos de receita e lucro ao longo do tempo
* Cálculo de indicadores financeiros
* Comparação entre empresas
* Ranking de empresas por lucro

## Tecnologias utilizadas

* Python
* Pandas
* Matplotlib
* Streamlit
* Jupyter Notebook

## Empresas analisadas

* Petrobras
* Vale
* Itaú
* Banco do Brasil
* Ambev
* Weg
* Magazine Luiza
* Bradesco

## Como executar o projeto

1. Clone o repositório
```
git clone https://github.com/seuusuario/b3-financial-analysis.git
```

2. Entre na pasta do projeto
```
cd b3-financial-analysis
```

3. Instale as dependências
```
pip install -r requirements.txt
```

4. Execute o dashboard
```
python -m streamlit run dashboard/app.py
```

O dashboard será aberto no navegador.
