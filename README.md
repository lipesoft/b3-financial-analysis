# B3 Financial Analysis

Projeto de análise de empresas listadas na B3 utilizando Python e dados públicos da CVM.

## Objetivo

Analisar indicadores financeiros como:

- Receita
- Lucro
- Margem líquida
- Crescimento de receita

## Tecnologias

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Estrutura

data → dados financeiros  
src → scripts de análise  
notebooks → exploração de dados
